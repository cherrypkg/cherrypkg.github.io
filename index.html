Speed up your programming in Red with Cherry.

Cherry is an extremely powerful package manager for the Red Programming Language.


Get Started

Get Started

Getting started with Cherry is extremely simple.

You'll want to begin by running git clone https://github.com/cherrypkg/cherry in a directory of your liking.

Then, build Cherry by using the prod.bat, which is located in the /build folder. If you're not on Windows, you can still run the commands inside of the files — Red's CLI is system agnostic.

The Red compiler will ask you if you want to build Cherry in dev or in prod. Simply type prod and press enter.
